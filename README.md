## ✨ About the Project

Welcome to the **Gacha Pity Simulator**! Ever wondered how much premium currency you need to guarantee your favorite character? This app simulates the gacha pulling experience, complete with pity mechanics, 50/50 rates, and drop probabilities, all without spending a single dime of real money!

## 🚀 Features

* 🎲 **Realistic RNG:** Accurately mimics standard gacha drop rates.
* 📊 **Pity Tracking:** Visualizes Soft Pity and Hard Pity thresholds so you know exactly when you'll strike gold.
* ⚖️ **50/50 System:** Accurately simulates winning and losing the 50/50 rate-up.
* ⚡ **Fast & Responsive:** A smooth UI/UX built for quick and satisfying simulated pulls.
